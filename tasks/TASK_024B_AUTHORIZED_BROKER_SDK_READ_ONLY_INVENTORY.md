# TASK_024B：用户已授权券商SDK包与只读行情能力盘点

状态：NEXT

## 目标

基于用户本机实际存在的官方券商客户端、SDK安装包、开发文档和账户权限，选择首个可以合法接入的只读行情或参考数据Provider。

## 优先候选

1. 中国银河证券星耀数智或TFast官方授权包；
2. 中泰证券XTP Quote/Level2官方包；
3. 国信证券iQuant当前客户端及内置开发文档；
4. 广发证券投易通官方量化组件；
5. 用户实际持有的其他券商官方SDK。

排序不代表用户已获得授权。真实证据优先于目录排序。

## 只读门禁

- 必须能够在不初始化交易会话的情况下发现或使用行情能力；
- 行情与Trader/Order/Cancel/Fund Transfer模块必须隔离；
- 不保存账户、密码、Token、IP、端口或秘密值；
- 不发送订单；
- 不修改券商客户端或注册表；
- 不自动安装大体积组件；
- SDK版本和Python兼容性必须来自官方文档或本机官方包；
- 输出字段必须对照交易所官方语义目录；
- 进入项目前必须经过Canonical、血缘、时点和Readiness门禁。

## 输出

- 本机官方券商产品和SDK证据清单；
- 授权状态与需用户确认的非秘密信息；
- 只读行情能力矩阵；
- SDK/客户端/Python/Windows兼容矩阵；
- 首个正式券商只读Provider候选；
- 薄适配任务书；
- 交易能力阻断证明。
