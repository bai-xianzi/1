# TASK_024：官方交易所与券商接口基线

## 1. 任务定位

TASK_024纠正此前“先以第三方聚合源作为首个外部Provider”的方向。

项目必须先以交易所、监管和结算机构的官方规则及接口文档建立语义基准，再以用户已经获得授权的券商官方SDK或数据接口作为个人项目的优先实际接入通道。AKShare、Tushare等第三方来源只能承担补充研究、发现线索和交叉核验，不得成为Canonical事实源或首个正式外部Provider。

## 2. 两条优先级

### 2.1 语义权威优先级

```text
交易所、监管、结算机构官方资料
> 已授权券商官方数据接口
> 已授权商业数据厂商官方SDK
> 第三方聚合与订阅服务
> 未验证网页接口或逆向爬取
```

### 2.2 个人项目实际接入优先级

```text
用户已授权的券商官方SDK或接口
> 可合法获得的交易所官方文件、历史数据或数据服务
> 用户已授权的商业数据厂商官方SDK
> 第三方聚合源，仅补充用途
```

交易所直连行情通常面向会员、市场参与者、信息服务商或持牌数据客户。项目应以其官方规范作为字段、状态、消息、时间和市场规则的权威基准，但不能假定个人用户已经拥有直连权限。

## 3. TASK_024A：官方接口目录与来源权威门禁

状态：COMPLETED_BY_DELIVERY_PACKAGE_PENDING_LOCAL_ACCEPTANCE

交付：

- `source_authority_policy_v1.json`：来源权威分级；
- `official_interface_catalog_v1.json`：首批官方交易所、结算机构、券商和商业SDK目录；
- `source_authority.py`：离线权威门禁与排序；
- 自动测试和离线报告；
- 对TASK_023C中AKShare优先结论的纠正；
- 项目架构图当前位置更新。

TASK_024A不做：

- 不安装任何SDK；
- 不登录任何券商或商业终端；
- 不调用交易所、券商、商业厂商或第三方网络接口；
- 不保存账户、密码、Token或安装路径；
- 不激活交易执行；
- 不把公开产品介绍误认为已经获得SDK授权。

## 4. 首批官方接口基线

### 4.1 市场基础设施语义基准

- 上海证券交易所：Binary、STEP、行情文件交换及技术支持体系；
- 深圳证券交易所：数据接口门户、STEP/Binary、行情网关和文件接口；
- 北京证券交易所及全国股转系统：交易、行情和文件接口规范；
- 香港交易所：OMD-C证券市场数据流、开发指南、连接指南及历史数据下载API；
- 中国结算：接口规范、技术资料、测试资料及登记结算规则。

### 4.2 已核验的券商官方候选

- 中国银河证券：星耀数智、TFast；
- 中泰证券：XTP Quote/Level2/Trader及Python API；
- 国信证券：iQuant Python、行情、财务与交易能力；
- 广发证券：投易通的实时行情、Python量化分析和量化交易能力。

这些产品的“存在”与“用户已经获得使用权”是两件事。下一任务必须检查用户持有的官方安装包、开发文档、账户权限和只读行情能力，不能凭产品页面直接开发。

## 5. 第三方来源边界

AKShare和Tushare：

- 不作为官方事实基准；
- 不作为首个正式外部Provider；
- 不用于决定交易日历、证券状态、公司行为或成交规则的最终语义；
- 不允许通过未经授权的高频抓取进入生产链路；
- 必须单独审查上游网页条款、频率限制、接口稳定性和IP封禁风险；
- 最多用于研究补充、发现线索和跨来源异常提示。

## 6. 交易能力隔离

券商SDK常同时提供行情与交易能力。TASK_024阶段只允许：

```text
官方文档读取
→ 能力发现
→ 只读行情或参考数据评审
→ Canonical映射设计
→ Readiness门禁设计
```

交易登录、报单、撤单、资金划转、账户操作和自动执行全部留在独立的交易安全项目中。

## 7. 下一任务

### TASK_024B：用户已授权券商SDK包与只读行情能力盘点

优先检查：

1. 用户实际拥有哪一家券商的官方客户端、SDK、开发文档或量化权限；
2. 是否存在只读行情接口，可以完全不初始化交易会话；
3. SDK版本、Python版本、Windows架构和依赖；
4. 行情字段、市场状态、证券主数据、交易日历和公司行为能力；
5. 官方文档与交易所语义基准的映射关系；
6. 许可证、账户授权、连接和测试环境要求；
7. 行情与交易模块是否可以物理隔离。

如果用户没有任何券商SDK授权，则先建立“官方文档语义目录与可下载官方文件”适配，不退回AKShare作为主源。
