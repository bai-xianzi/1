# TASK_020A 全供应商适配与单机资源合同

## 目标

将以下要求转化为项目权威约束和机器可读合同：

```text
不绑定Wind
不绑定iFinD
不绑定银河证券星耀数智
不绑定任何单一券商或数据源
```

所有来源必须通过供应商无关的能力模型进入StandardDataService。

## 交付

- 全供应商多源适配架构权威文件；
- Provider能力目录；
- Provider目标矩阵；
- 供应商能力发现状态；
- 当前Windows单机资源档案；
- 小批次、低并发、磁盘预检与断点原则；
- 权威文件同步标记；
- 验证器和测试。

## 当前实现边界

本任务不安装或调用商业SDK，不保存账号、Token或密码，不下载大型数据，不修改DolphinDB。
