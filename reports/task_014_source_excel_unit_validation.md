# TASK_014 源Excel单位核验

来源文件：

```text
2026-06-19更新简化个股基本面数据.xlsx
```

## 结论

来源 Excel 和 DolphinDB Raw 层的数值保持原样，不做回写或修正。

```text
股本字段：
来源单位 = 万股
Canonical = 来源值 × 10000 → 股

金额字段：
来源单位 = 千元人民币
Canonical = 来源值 × 1000 → 人民币元

每股字段：
来源单位 = 元/股
Canonical = 原值，不乘倍数
```

这里的倍数是 **Raw→Canonical 单位标准化**，不是认为源 Excel 数值有错误。

## 平安银行 000001 样例

| 字段 | Raw来源值 | Canonical值 |
|---|---:|---:|
| 总股本 | 1,940,591.87 万股 | 19,405,918,700 股 |
| 营业收入 | 35,277,000 千元 | 35,277,000,000 元 |
| 每股收益 | 0.67 元/股 | 0.67 元/股 |
| 调整每股净资产 | 23.91 元/股 | 23.91 元/股 |

所有 Raw 原值在 DolphinDB 中保留；查询时请求 `include_source_extensions=True`
还会在标准记录中返回 Raw 原值和单位合同。
