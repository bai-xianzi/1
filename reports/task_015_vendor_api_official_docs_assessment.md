## Wind公开官方资料可支持的设计

Wind Client API公开资料确认其面向桌面或第三方应用，
支持公司和行业分析、历史日线、Tick、分钟回放、宏观时间序列、
投资组合和风险信息，并支持多种编程语言和代码生成器。

因此本项目只预定义能力，不在业务代码写死Wind字段和函数。
未来开通后，使用Wind代码生成器形成版本化VendorQueryManifest。

## iFinD公开官方资料可支持的设计

iFinD公开Python/HTTP示例包含：

- access token；
- 高频序列；
- 实时行情；
- 历史行情；
- 基础数据；
- 日期序列；
- 数据池；
- 经济数据库；
- 日内快照；
- 公告查询；
- 智能选股；
- 交易日查询。

官方帮助说明不同账号权限不同、单次请求存在数据量限制，
且完整指标命令需要通过超级命令工具生成。

因此本项目：

- 不在代码内保存refresh token和access token；
- 不把公开示例中的指标当成永久标准字段；
- 把指标、参数、报表ID和返回路径放入版本化查询清单；
- 配额仅作为账号级运行配置，不写死在业务层；
- 开通后先做权限和真实返回验收，再允许启用。

## 长期接入原则

```text
Wind / iFinD / 券商 / DolphinDB / 文件
→ 来源适配器
→ RawDataBatch
→ 来源映射与语义适配
→ Canonical标准对象
→ 质量、血缘、可见时间和冲突记录
→ StandardDataService
→ 下游
```

券商交易执行能力不放入市场数据适配器。
未来订单、成交、持仓和账户能力必须使用独立执行合同。
