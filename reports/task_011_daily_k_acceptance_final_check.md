## 序列化修订

首次报告中的来源空值被 Python JSON 编码器写为 `NaN`。
`NaN` 不是严格 JSON 标准值，部分语言和数据平台会拒绝解析。

修订后：

- 非有限浮点数统一输出为 `null`；
- `json.dumps(..., allow_nan=False)` 强制阻止非法JSON；
- 不修改 DolphinDB 原始值；
- 不影响验收结论和标准对象内容。
