# 项目长期记忆逻辑复核报告（V8）

复核日期：2026-06-22

## 一、本轮纳入的长期原则

### 从大型量化机构借鉴

- 研究生产线而非单一策略；
- 实验注册和可复现；
- 失败实验留档；
- 因子、模型和策略生命周期；
- 样本外、Walk-forward、交易成本和容量；
- CWMS合成市场和极端场景验证；
- Alpha、组合、风险和执行共同决定最终结果。

### 从券商借鉴

- 策略与订单执行分离；
- 独立交易前风控；
- 订单状态机；
- 幂等防重复；
- 分级停机；
- 订单、成交、持仓、现金和费用账实核对；
- RESEARCH、BACKTEST、PAPER和LIVE环境隔离；
- 部署、备份、恢复和版本回滚。
