# 项目长期记忆逻辑复核报告（V6）

复核日期：2026-06-22

## 本轮核心修正

V5已经正确建立用户场景控制平面，但自然语言场景仍可能被误解为需要新增同名变量。本轮将领域模型进一步抽象为：

```text
WorldEntity
+ EntityRelation
+ WorldEvent
+ EventImpact
+ EntityExposure
+ CausalRule
```

## 已解决的问题

1. 0/1门控与完整影响描述混淆；
2. 支柱产业结构关系与支柱产业受损动态事件混淆；
3. 仓库、工厂、港口等不同对象和不同灾害组合可能导致字段爆炸；
4. 用户举例可能被Codex机械转成字段；
5. 场景干预合同只能改字段值，不便于创建结构化事件和影响。

## 设计结论

- 布尔值只回答“有没有发生”；
- 枚举回答“发生了什么、影响了什么”；
- 数值和区间回答“影响多大”；
- 时间字段回答“持续多久、多久恢复”；
- 实体关系和暴露回答“为什么影响到该公司或行业”；
- 因果规则回答“后续变量如何联动”。
